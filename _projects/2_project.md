The Laplacian Library provides a centralized repository of Compressed Sparse Column (CSC) formatted Graph Laplacians, facilitating advanced analysis in power systems and graph signal processing (GSP).

This project aims to provide accessible and standardized graph data for the GSP research community.
- Source code on [GitHub](https://github.com/lukelowry/laplib)

The library is free to use to accelerate your research.
